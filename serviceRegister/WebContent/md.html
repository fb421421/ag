注册模块接口说明

1.获取验证码图片
请求地址：/serviceRegister/checkCode
请求方法：GET
URL参数：无
Data参数：无
成功返回：验证码图片
错误返回：异常页面
调用示例：/serviceRegister/checkCode
注意事项：无

2.注册会员
请求地址：/serviceRegister/rest/user
请求方法：POST
URL参数：无
Data参数：{"userName":"nu2222ll","password":"111222333","checkCode":"6229"}
成功返回：HTTP CODE:200 Content:{"status":1,"errorMessage":null}
错误返回：HTTP CODE:200 Content:{"status":0,"errorMessage":"error message"}
调用示例：$.ajax({ url: "/serviceRegister/rest/user", dataType: "json", data :{"userName":"nu2222ll","password":"111222333","checkCode":"6229"} , type : "POST", success : function(r) { console.log(r); } });
注意事项：无
